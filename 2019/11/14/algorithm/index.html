LCS Longest Common Subsequence

  • 运用动态规划方法查找给定两个序列的最大公共子序列（子序列之间的字符可以不连续，若子串substring的字符必须连续）
  • 由序列尾部开始：
    • 若两个序列尾部字符相等：LSC[i][j] = LSC[i-1][j-1] +1
    • 若两个序列尾部字符不相等：LSC[i][j] = max(LSC[i-1][j], LSC[i][j-1])
