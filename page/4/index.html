spring学习之工具篇

web学习 之 spring框架

  • IDE
  • Spring-tool-suite
  • Spring for framework

1、IDE 的选择

  • Myeclipse

    MyEclipse Web 开发工具支持快速添加技术功能到 Web 项目中，使用可视化编辑器以便编码和配置，并且在多种应用服务器上测试你的任务。
    Myeclipse 拥有大量功能强大的“收费”插件(包括 Spring 插件，因此有了 Myeclipse 则可跳过安装 Spring-tool-suite )，但几乎个版本都有破解。
    官方下载
    2016 ci7 Myeclipsep + 破解软件 密码: 34ax
    破解方式

  • Eclipse for JavaEE

    Eclipse 是一个开放源代码的、基于Java的可扩展开发平台。就其本身而言，它只是一个框架和一组服务，用于通过插件组件构建开发环境。
    Eclipse 有多种版本，可直接下载 JavaEE Web版本
    Eclipse 经典版本则需要下载JavaEE插件
    1. 打开 Eclipse -> Help -> Install New SoftWare
    2. 在窗口的 work with 中输入 http://download.eclipse.org/releases/neon (neon是Eclipse版本名) 注：下载路径一定要与版本号相对应，否则会安装过慢甚至安装失败。
    3. 勾选以下4个选项，然后 next 到安装完重启。
      • Eclipse Java EE Developer Tools
      • Eclipse Java Web Developer Tools
      • Eclipse Web Developer Tools
      • Eclipse XMl Editors and Tools

  • IDEA UItimate

    IntelliJ IDEA 是java语言开发的集成环境，IntelliJ在业界被公认为最好的java开发工具之一，尤其在智能代码助手、代码自动提示、重构、J2EE支持、各类版本工具(git、svn、github等)、JUnit、CVS整合、代码分析、 创新的GUI设计等方面的功能可以说是超常的。
    IntelliJ IDEA UItimate 是付费版，集合了各种强大插件，而社区免费版没有，但可以尝试。
    官方下载

2、Spring-tool-suite

  • Spring-tool-suite 在 MyEclipse 打开
    • 右击项目 -> MyEclipse -> Project Facets -> install spring facet
  • Spring-tool-suite 主要安装与Eclipse
    • Windows：
      1. 打开 Eclipse -> Help -> Install New SoftWare
      2. Add -> Archive -> 选中下载好的 Spring-tool-suite 插件zip (必须对应当前使用的Eclipse版本)
      3. 导入安装

    • Linux:

      1. Spring-tool-suite 需要下载Linux对应的 tar.gz

      2. 打开终端

      3. 即可在 Eclipse 中打开 Spring IDE

3、Spring for framework

  • Spring for framework 在那个IDE用处都一致
    1. 下载所需要的 Spring for framework 版本
    2. 将解压目录下 libs 的 jar 复制项目新建的 lib 文件夹中 (至少要有4个基本jar包)
      • spring-beans-4.3.1.RELEASE.jar
      • spring-context-4.3.1.RELEASE.jar
      • spring-core-4.3.1.RELEASE.jar
      • spring-expression-4.3.1.RELEASE.jar
    3. 右击项目 -> Build Path ->Configure Build Path -> 将 lib 中的 jar 包全部加入 -> apply and close
